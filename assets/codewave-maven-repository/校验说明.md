# CodeWave Maven 内置仓库

本目录按 Maven 标准仓库结构保存 CodeWave 元数据插件和采集器，解决它们不在 Maven Central、阿里云公共仓库中导致的构建失败。

当前坐标：

```text
com.netease.lowcode:nasl-metadata-maven-plugin:1.7.6
com.netease.lowcode:nasl-metadata-collector:0.17.0
```

网易官方公开案例同样采用随仓库提供插件 JAR/POM 和安装脚本的方式：

- https://github.com/netease-lcap/libraryDemo/tree/master/jar
- https://github.com/netease-lcap/codewave-architect-course/tree/main/example/java_lib/jar
